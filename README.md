# availability-bot
Telegram bot to check and change availability of two people. 
Adapted from [Echo Bot](https://docs.python-telegram-bot.org/en/stable/examples.echobot.html).

# Requirements
```python==3.9```
```python-telegram-bot==20.0a2```
